Modern Discord Bots

Zap and Bolt are modern Discord bots for a modern era. They're both written using Discord.rb. Zap provides a ton of cool, fun features, and Bolt provides moderation commands that server owners will find super useful. The bots also interact together well, and more features for this are a WIP (being added very soon)! Invite to your server today! Since they're open source, you can also host them yourself.
